# OWASP Top 10 Mapping for Microsoft 365

This document maps common Microsoft 365 features and use cases to the OWASP Top 10 security risks. It helps identify potential vulnerabilities and provides guidance for mitigation.

---

| OWASP Risk | Microsoft 365 Feature | Use Case | Risk Description | Mitigation |
|------------|------------------------|----------|------------------|------------|
| [A01: Broken Access Control](https://owasp.org/Top10/A01_2021-Broken_Access_Control/) | OneDrive | Sharing files via "Anyone with the link" | Unauthorized access to sensitive files if the link is leaked or guessed | Restrict sharing to specific individuals. Microsoft guidance [here](https://learn.microsoft.com/en-us/sharepoint/shareable-links-anyone-specific-people-organization) |
| [A02: Cryptographic Failures](https://owasp.org/Top10/A02_2021-Cryptographic_Failures/) | Outlook / SharePoint | Sending or storing sensitive data without encryption | Emails or documents may be intercepted or accessed without protection | Use built-in encryption (e.g. Sensitivity Labels, TLS) and avoid sending personal or sensitive data unprotected. Microsoft guidance [here](https://learn.microsoft.com/en-us/purview/sensitivity-labels) and [here](https://learn.microsoft.com/en-us/purview/prepare-tls-1.2-in-office-365) |
| [A03: Injection (includes LLM01:2025 Prompt Injection)](https://owasp.org/Top10/A03_2021-Injection/) | Copilot (Outlook) | Prompt includes "Ignore previous instructions" | AI may execute unintended actions based on manipulated input | Sanitize prompts and restrict Copilot access to sensitive data. Microsoft guidance [here](https://learn.microsoft.com/en-us/copilot/microsoft-365/microsoft-365-copilot-ai-security) |
| [A04: Insecure Design](https://owasp.org/Top10/A04_2021-Insecure_Design/) | Teams | Adding external guests to internal teams | Guests may access internal chats and files beyond intended scope | Apply guest access policies and review team permissions regularly. Microsoft guidance [here](https://learn.microsoft.com/en-us/microsoftteams/guest-access) |
| [A05: Security Misconfiguration](https://owasp.org/Top10/A05_2021-Security_Misconfiguration/) | Outlook | Auto-forwarding emails to external domains | Misconfigured rules may leak internal data to outside recipients | Disable external forwarding and review mailbox rules regularly. Microsoft guidance [here](https://learn.microsoft.com/en-us/defender-office-365/outbound-spam-policies-external-email-forwarding)|
| [A06: Vulnerable and Outdated Components](https://owasp.org/Top10/A06_2021-Vulnerable_and_Outdated_Components/) | Office Apps (Excel, Word) | Using outdated desktop versions without security patches | Older applications may lack critical security updates or exploit mitigations | Enable auto-updates and enforce minimum version policies. Microsoft guidance [here](https://learn.microsoft.com/en-us/microsoft-365-apps/updates/overview-update-channels) |
| [A07: Identification and Authentication Failures](https://owasp.org/Top10/A07_2021-Identification_and_Authentication_Failures/) | Microsoft Account / Azure AD | Weak passwords or lack of multi-factor authentication (MFA) | Unauthorized access to accounts due to poor authentication controls | Enforce MFA, password policies, and conditional access. Microsoft guidance [here](https://learn.microsoft.com/en-us/entra/identity/conditional-access/overview) and [here](https://learn.microsoft.com/en-us/microsoft-365/admin/security-and-compliance/set-up-multi-factor-authentication?view=o365-worldwide) |
| [A08: Software and Data Integrity Failures](https://owasp.org/Top10/A08_2021-Software_and_Data_Integrity_Failures/) | Power Automate | Using unverified connectors or scripts | Malicious or untrusted flows may manipulate or exfiltrate data | Use approved connectors only and review flow permissions. Microsoft guidance [here](https://learn.microsoft.com/en-us/power-platform/admin/wp-data-loss-prevention) and [here](https://learn.microsoft.com/en-us/power-platform/admin/admin-documentation?tabs=new) |
| [A09: Security Logging and Monitoring Failures](https://owasp.org/Top10/A09_2021-Security_Logging_and_Monitoring_Failures/) | Microsoft 365 Admin Center | Lack of audit logging or alerting | Suspicious activity may go unnoticed without proper monitoring | Enable audit logs, alerts, and use Microsoft Secure Score recommendations. Microsoft guidance [here](https://learn.microsoft.com/en-us/purview/audit-search) and [here](https://learn.microsoft.com/en-us/defender-xdr/microsoft-secure-score) |
| [A10: Server-Side Request Forgery (SSRF)](https://owasp.org/Top10/A10_2021-Server-Side_Request_Forgery_%28SSRF%29/) | Forms / Power Automate | Automated requests triggered by user input | Malicious input may cause backend services to access unintended resources | Validate all input and restrict external request capabilities in flows and forms. Microsoft guidance [here](https://learn.microsoft.com/en-us/power-automate/add-manage-connections) and [here](https://learn.microsoft.com/en-us/power-platform/admin/wp-data-loss-prevention) |
